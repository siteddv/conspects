## Краткое содержание

Глава представляет практическую работу по созданию хранилища данных на основе методологии Data Vault. Выполняется настройка среды и подключение к базе данных, создание и наполнение staging-слоя для хранения сырых данных. Рассматривается создание основных объектов Data Vault: хабов (Hubs) для хранения бизнес-ключей, линков (Links) для связей между хабами и сателлитов (Satellites) для хранения описательных атрибутов. Разбираются практические нюансы методологии Data Vault, выполняется разбор домашнего задания. В конце приводится код, созданный в ходе занятия.
