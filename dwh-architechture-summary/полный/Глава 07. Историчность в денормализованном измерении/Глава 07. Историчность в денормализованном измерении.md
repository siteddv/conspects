# Глава 07. Историчность в денормализованном измерении

Седьмая глава курса посвящена поддержке историчности в денормализованных измерениях. Рассматриваются особенности работы с историчностью в измерениях, которые содержат денормализованные данные, методы отслеживания изменений и создания новых версий записей.

В денормализованных измерениях историчность поддерживается через добавление полей `valid_from` и `valid_to` для определения периода действия записи. При изменении атрибута создается новая запись с обновленными данными и новым периодом действия. Старая запись остается в базе для исторического анализа.
