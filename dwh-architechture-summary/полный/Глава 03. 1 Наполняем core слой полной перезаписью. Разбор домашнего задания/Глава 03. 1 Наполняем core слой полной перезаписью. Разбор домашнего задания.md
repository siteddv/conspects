## Краткое содержание

Глава начинается с создания хранимой процедуры для загрузки данных в таблицу фактов `fact_rental`. Процедура выполняет удаление старых данных, получение актуальных данных из стейджинг-слоя, обработку данных (преобразование дат, вычисление сумм платежей) и присоединение измерений.

Рассматривается пересоздание стейджинг- и core-слоев для обеспечения корректной структуры хранилища. Выполняется добавление новых строк в таблицу rental и тестирование процесса загрузки данных для проверки корректности работы процедур.
