## Краткое содержание

Глава завершает практическую работу по созданию хранилища данных на основе методологии Data Vault. Выполняется настройка окружения и создание схемы DataVault. Рассматривается работа с суррогатными ключами и расчет хэшей для обеспечения уникальности и целостности данных. Выполняется создание и загрузка основных объектов Data Vault: хабов (Hubs) для хранения бизнес-ключей сущностей, линков (Links) для связей между сущностями и сателлитов (Satellites) для хранения описательных атрибутов с поддержкой историчности. В конце выполняется тестирование созданного хранилища, отладка процедур загрузки и обсуждение практических аспектов работы с Data Vault.
