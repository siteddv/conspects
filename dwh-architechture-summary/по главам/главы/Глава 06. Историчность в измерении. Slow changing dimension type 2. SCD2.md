# Глава 06. Историчность в измерении. Slow changing dimension type 2. SCD2

Шестая глава курса посвящена поддержке историчности данных в измерениях с использованием метода Slow Changing Dimension Type 2 (SCD2). Рассматривается отслеживание изменений атрибутов во времени, создание новых версий записей при изменениях, определение периодов действия записей.

SCD2 — это метод поддержки историчности в измерениях, при котором при изменении атрибута создается новая запись с новым периодом действия, а старая запись помечается как неактивная или получает дату окончания действия. Это позволяет отслеживать, как изменялись данные во времени, и строить отчеты на исторических срезах.

В измерениях добавляются поля `valid_from` (дата начала действия записи) и `valid_to` (дата окончания действия записи, NULL для текущей версии). При изменении атрибута создается новая запись с обновленными данными и новым периодом действия. Старая запись остается в базе для исторического анализа, но получает дату окончания действия.

Рассматриваются различные подходы к определению изменений: сравнение всех атрибутов, использование хэшей для определения изменений, отслеживание изменений через триггеры в источнике. Создаются процедуры для обработки изменений с поддержкой историчности, которые проверяют наличие изменений и создают новые версии записей при необходимости.
