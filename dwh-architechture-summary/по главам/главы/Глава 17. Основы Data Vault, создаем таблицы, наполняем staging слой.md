# Глава 17. Основы Data Vault, создаем таблицы, наполняем staging слой

Семнадцатая глава курса посвящена практической реализации Data Vault. Рассматривается создание staging слоя для Data Vault, разработка процедур для наполнения staging слоя, частичное создание таблиц Data Vault слоя.

Начинается практическая работа с Data Vault. Создается staging слой — промежуточный слой для загрузки сырых данных. Разрабатываются процедуры для наполнения staging слоя. Создаются первые таблицы Data Vault слоя: хабы, линки и сателлиты.
