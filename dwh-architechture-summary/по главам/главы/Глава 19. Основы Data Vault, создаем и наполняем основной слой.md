# Глава 19. Основы Data Vault, создаем и наполняем основной слой

Девятнадцатая глава курса завершает изучение Data Vault практической реализацией основного слоя. Рассматривается загрузка всех данных в Data Vault, создание процедур наполнения Data Vault, реализация витрин данных.

Загружаются все данные в Data Vault. Создаются процедуры наполнения Data Vault для загрузки данных в хабы, линки и сателлиты. Реализуются витрины данных (Data Marts) на основе Data Vault слоя. Добавляются визуализации и финализируется Data Vault хранилище.
