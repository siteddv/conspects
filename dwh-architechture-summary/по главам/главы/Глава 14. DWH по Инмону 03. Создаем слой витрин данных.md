# Глава 14. DWH по Инмону 03. Создаем слой витрин данных

Четырнадцатая глава курса завершает изучение методологии Инмона созданием слоя витрин данных. Рассматривается создание витрин на основе DDS слоя для конкретных бизнес-задач.

Витрины данных создаются на основе DDS для конкретных бизнес-задач. Витрины содержат агрегированные данные, готовые для анализа, и оптимизированы под конкретные запросы. Создаются процедуры для загрузки данных из DDS в витрины с агрегацией и трансформацией.
