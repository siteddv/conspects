### Часть 2: Обзор приложения «Книжный магазин 2.0»

#### Общее описание приложения:
Приложение "Книжный магазин 2.0" представляет собой обновлённую версию приложения из курса Dev-1 с расширенным функционалом для работы с большим количеством пользователей и книг.

#### Основные компоненты:
1. **Интернет-магазин для покупателей**:
   - Поиск и просмотр каталога книг.
   - Детальная информация о книгах.
   - Система корзины для зарегистрированных пользователей.
   - Возможность покупки книг.

2. **Административная панель**:
   - Управление поступлением книг от поставщиков.
   - Установка розничных цен на книги.
   - Система фоновых задач для выполнения длительных операций.

#### Архитектура приложения:
- Все взаимодействие с базой данных происходит через специальные функции.
- Функции разделены по схемам: `web` для интернет-магазина и `admin` для админки.
- Используется система аутентификации на основе токенов.

#### Особенности реализации:
- Фоновые задачи позволяют сотрудникам запускать длительные операции и проверять результаты позже.
- Информационная панель показывает вызываемые функции, параметры и время выполнения запросов.
