## Введение и обзор типов данных

PostgreSQL позволяет создавать пользовательские типы данных, которые расширяют возможности системы. Пользовательские типы данных могут быть созданы на основе существующих типов или полностью определены пользователем.

Основные категории пользовательских типов данных: составные типы (composite types) — набор атрибутов с собственными именами и типами данных, перечисления (enums) — набор именованных значений, диапазонные типы (range types) — типы для представления диапазонов значений и домены (domains) — типы с ограничениями на основе существующих типов.

## Составные типы и перечисления

Составные типы данных представляют собой набор атрибутов (полей), каждый из которых имеет собственное имя и тип данных. Эти типы аналогичны структурам данных в других языках программирования. Составные типы могут использоваться в таблицах, функциях и операциях.

Перечисления (enums) позволяют создавать набор именованных значений. Это полезно для ограничения возможных значений столбца определенным набором опций.

## Диапазонные типы и интервалы

Диапазонные типы позволяют представлять диапазоны значений, такие как временные интервалы, числовые диапазоны и другие. PostgreSQL предоставляет встроенные диапазонные типы: `int4range`, `int8range`, `numrange`, `tsrange`, `tstzrange`, `daterange`.

## Домены и пользовательские типы

Домены — это типы данных с ограничениями на основе существующих типов. Они позволяют создавать специализированные типы с дополнительными проверками и ограничениями. Домены полезны для обеспечения целостности данных на уровне типов.
